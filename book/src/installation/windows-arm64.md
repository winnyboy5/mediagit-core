# Windows ARM64 Installation

Install MediaGit on Windows ARM64 (Surface, Copilot+ PCs).

## Download

Download from [Releases](https://github.com/yourusername/mediagit-core/releases):
- `mediagit-windows-arm64.exe`

## Install

```powershell
# Add to PATH or place in accessible location
mv mediagit-windows-arm64.exe $env:USERPROFILE\bin\mediagit.exe
```

## Verify

```powershell
mediagit --version
```
